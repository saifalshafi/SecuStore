About SecuStore

SecuStore is a secure file storage platform focused on privacy and protection. It helps users upload, store, and share files safely with encryption and strong authentication.

Why SecuStore?
  • AES-based encrypted file storage
  • Email-based One-Time Password (2FA)
  • Real-time activity logging using Django Auditlog
  • Brute-force protection
