Description of Service

SecuStore provides a web-based encrypted file storage and management service. All files uploaded to the Platform are encrypted using AES-256-CFB before being written to storage. The Platform also provides:

  • File upload, download, and deletion with full audit logging
  • File metadata management (descriptions, tags, categories, permissions)
  • File expiration with automatic deletion
  • Two-Factor Authentication (2FA) via email OTP for all user accounts
  • An immutable blockchain-based audit ledger recording all file operations

Data Privacy & Security

SecuStore is committed to protecting your data. The following security measures are applied to all accounts and files:

  • Encryption at rest: All files are encrypted with AES-256-CFB. Plaintext files are never stored on the server filesystem.
  • Integrity verification: HMAC-SHA256 signatures are verified before every file download. Any tampered file will be rejected.
  • Authentication: Mandatory Two-Factor Authentication (OTP) is required for every login. Passwords are hashed using Argon2.
  • Audit logging: All file operations are permanently recorded in a tamper-evident blockchain ledger.
  • Brute-force protection: Accounts are locked after three consecutive failed login attempts.

Despite these measures, no system can guarantee absolute security. You acknowledge and accept the inherent risks of transmitting data over the internet.

File Deletion & Data Retention

You may delete any of your files at any time. Deletion is immediate and permanent — encrypted file binaries are removed from disk and all associated database records are destroyed. Files with an expiration date set will be automatically and permanently deleted on that date. We do not maintain backups of user-deleted files.
